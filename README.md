# Java Technical Documentation Page

A responsive, clean, and modern Technical Documentation Page focusing on core Java fundamentals. This project showcases structured semantic HTML5 layout practices combined with advanced CSS techniques like Grid and Flexbox for seamless cross-device adaptability.

Developed as part of the **freeCodeCamp Responsive Web Design** curriculum.

## 🚀 Live Demo

You can view the live site here: [https://telesphore8527.github.io/java-technical-docs/]

## 🛠️ Features

- **Split-Screen Desktop Layout:** Side-by-side view featuring a fixed, scrollable navigation bar on the left and comprehensive technical content on the right.
- **Fully Responsive Architecture:** Employs CSS media queries to dynamically adjust layouts for mobile devices (`max-width: 760px`) and larger screens.
- **Smooth Navigation:** Fluid, anchor-link page scrolling (`scroll-behavior: smooth`) for enhanced user experience.
- **Semantic Structure:** Leverages modern HTML5 elements (`<nav>`, `<main>`, `<section>`, `<header>`) alongside properly formatted code blocks (`<pre><code>`) for technical snippets.

## 🧰 Tech Stack

- **HTML5:** Semantic markup, hyperlinking, and structured categorization.
- **CSS3:** Native CSS Grid, Flexbox, layout adjustments, custom scrollbars, and media queries for mobile responsiveness.

## 📂 Project Structure

## 📖 Content Covered
This documentation page covers the structural building blocks of Java programming:
1. Introduction & "Write Once, Run Anywhere" paradigm  
2. Core Technical Prerequisites  
3. Architectural differences between Java and JavaScript  
4. Standard Hello World application breakdown  
5. Variable declarations and naming conventions  
6. Primitive vs. Reference Data Types  
7. Control Flow Statements (`if-else`)  
8. Loops and Iteration (`for`, `while`, `do-while`)  
9. Method / Function syntax and declarations  
10. Official Reference documentation links
