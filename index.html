Introduction

Java is an object-oriented programming language created by James Gosling at Sun Microsystems in 1995. It is now maintained by Oracle Corporation. Java was designed to be portable, secure, and robust. Thanks to the Java Virtual Machine (JVM), Java programs can run on different operating systems without modification. This concept is often summarized by the phrase Write Once, Run Anywhere.

Java is widely used to build:

  • Web applications
  • Mobile application(especially android)
  • entreprise software
  • Distributed software

Its popularity comes from its reliability, large ecosystem, and strong community

What you should already know

Before starting with Java, it is recommended to have some basic knowledge of computing and programming.

The main prerequisites include:

  • Basic computer knowledge:understanding how computers, files, and operating systems work.
  • Programming logic:concepts such as variables, conditions, loops, and algorithms.
  • Basic algorithmic thinking:the ability to break down a problem into logical steps.
  • Using a code editor or IDE such as IntelliJ IDEA, Eclipse IDE, or Visual Studio Code.
  • Basic technical English skills,since most documentation and resources are written in English.
Java and javascript

Despite the similarity in their names, Java and JavaScript are two different and independent programming languages.

ava is an object-oriented language mainly used to develop enterprise applications, Android mobile applications, and complex systems. It runs on the Java Virtual Machine (JVM), which allows programs to run on multiple platforms.

JavaScript, on the other hand, is mainly used for web development to create interactive and dynamic web pages. It runs directly in web browsers and can also be used on the server side through environments such as Node.js.

Hello world program

The Hello World program is traditionally the first program written when learning a programming language such as Java. It helps understand the basic structure of a program and verify that the development environment is correctly installed.

Here is a simple example:


    public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        } 
    }

Explanation

  • class HelloWorld:defines a class named HelloWorld. In Java, every program is based on classes.
  • public static void main(String[] args):this is the main method where program execution begins.
  • System.out.println():this statement prints a message to the console. When the program runs, it simply displays: Hello world!

This example introduces some fundamental Java concepts such as classes, methods, and console output.

Variables

In Java, a variable is a memory space used to store a value that can be used and modified during program execution.


In Java, variables are used as symbolic names that store values in a program. The name of a variable is called an identifier, and it must follow specific naming rules.

A Java identifier must start with a letter, an underscore (_), or a dollar sign ($). The following characters may also include digits (0–9). Since Java is case-sensitive, uppercase and lowercase letters are considered different.
Identifiers may also include Unicode characters.

However, identifiers cannot be Java reserved keywords such as class, int, or public.


Each variable must be declared with a data type that specifies the type of value it can store (such as numbers, characters, or text).

Example 

    int age = 20; 
    double price = 10.5; 
    char letter = 'A'; 
    String name = "John";
    In this example:
  • int stores integer numbers
  • double stores decimal numbers
  • char stores a single character
  • String stores text

Variables are therefore used to store, manipulate, and reuse data in a program.

Data types

In Java, data types define the kind of value a variable can store. They help the program understand how to interpret and manipulate data.

Java data types are generally divided into two main categories:

1. Primitive Types

Primitive types store simple values directly. The main ones include

  • int: integer numbers
  • double: decimal numbers
  • char: a single character
  • boolean: logical values (true or false)
  • float, long, short, byte: other numeric types

2. Non-primitive Types (Reference Types)


Non-primitive types store references to objects. Examples include:

  • String: text or sequence of characters
  • Arraya collection of elements of the same type
  • Class and Object: structures used in object-oriented programming

Data types are important because they determine how much memory is used and what operations can be performed on variables.

if-else statement

if and else statements are used to make decisions in a program by executing certain blocks of code only if a condition is true.

example


    int age = 18;
    if (age >= 18) {
       System.out.println("You are an adult.");
    } else {
       System.out.println("You are a minor.");
    }
  • if (condition): executes the block if the condition is true.
  • else: executes the block if the condition is false.

You can also use else if to test multiple conditions:


    if (age < 13) {
        System.out.println("Child");
    } else if (age < 18) {
        System.out.println("Teenager");
    } else {
        System.out.println("Adult");
    }

These structures allow you to control the program's flow based on different conditions.

loops statement

Loops allow you to repeat a block of code as long as a condition is true. They are useful for processing collections of data or performing repetitive tasks.

1. for loop

Used when the number of iterations is known in advance.


    for (int i = 0; i < 5; i++) { 
        System.out.println("i = " + i);
    }
  • int i = 0: initialization
  • i < 5: continuation condition
  • i++: increment after each iteration

2. while loop

Repeats the block of code as long as a condition is true. Ideal when the number of iterations is unknown.


   int i = 0;
   while (i < 5) {
       System.out.println("i = " + i);
       i++;
   }

3. do-while loop

Similar to while, but executes the block at least once before checking the condition.


    int i = 0; 
    do { 
        System.out.println("i = " + i);
        i++;
    } while (i < 5);

Loops are essential for automating repetitive actions and making code more efficient.

    public int add(int a, int b) {

    }
  • This function takes two numbers a and b and returns their sum.
  • return is used to send the value back to the calling program.

Calling a Function


    int result = add(5, 3);

Functions help make code clearer, modular, and reusable.
